### 2.2概率论
#### 2.2.1基本概念
P概率
基本原则：
独立性
乘法原理：P(a&b)=p(a)\*p(b)

### 2.2.2集中趋势公式
描述一组概率分布的中心
#### a.对应于总体结果
二项分布:在N次实验中成功X次的概率
公式：
```math
f(x)=p^x(1-p)^{n-x}*\frac{n!}{(n-x)!}
```

期望值/均值

```math

E[x]=\mu_{x}=\sum_{i=1}^{\infty}P(x=x_{i})*x_{i}
```
另一个公式
```math
E[x]=\mu_{x}=\int_{-\infty}^{+\infty}f(x)*xd_{x}
```
f(x)是x的连续性随机变量的概率分布

##### b.对应样本结果
样本平均值
```math
\overline X=\sum_{i=1}^n\frac{x_{i}}{n}
```
几何平均数（只能用于正的数值）
```math
G(x)=(\prod_{i=1}^{n}x_{i})^{1/n}
```
>几何平均数可用来衡量一个投资者的收益，比算术平均数更严谨
>收益指你在整个投资中赚的钱占本金的百分比。总收益率=收益率+1

#### 离散趋势指标
衡量参数的变化程度
##### 概念

* 方差

```math
\sigma^2
```

* 标准差的平方

```math
S^2
```
```math
\sigma=s^2
```

* 总体方差

```math
Var(x)=\sum_{i=1}^{\infty}p_{(x=x_{i})}(x_{i}-\mu_{x})^2=\sigma^2
```
    >参数变动越多，方差越大

* 样本方差

```math
S_{(x)}^2=\sum_{i=1}^{n}\frac{(x-\overline x)^2}{n}
```

    >一般人们期望高收益（希望收益的期望值很高），并且稳定
    >期望值越高越好，方差越小越好
    >方差代表风险

* 协方差
    衡量两个变量一起变动的情况
    
```math
Cor_{(x,y)}=\sum_{i=1}^{n}(x_{i}-\overline x)(y_{i}-\overline y)/n
```
    >一个正值的协方差表示两个变量同向变动
    >负值表示二者反向变动

* 相关性系数ρ
-1<=ρ<=1
```math
\rho=\frac{Cor_{(x,y)}}{S_{x}S_{y}}
```

* 正态分布![9e0c6b5d43eb26d34b419e363019845b.svg+xml](en-resource://database/4922:1)
    正态分布属于一种钟形曲线

```math
f(x)=\frac{1}{\rho\sqrt{2\pi}}e^{-\frac{(x-\mu)^2}{2\rho^2}}
```
>金融学注重长尾分布
>服从长尾分布的随机变量出现极端值得概率比较大
>许多投机性资产的收益都是服从长尾分布的
>极端值的出现不可预料，又名《黑天鹅现象》

* 现值PV
    资产在现在的价值
    >资金有时间价值
    >与之对应的为贴现值，或现期贴现值PDV
    >
```math
PDV=PV=1/(1+r)^{n}
```
    r利率，n为时间段

* 公债/永续年金
    无限期等额收付的特种年金。
    每次支付的利息称为券息或息票，c\*g
    假设每年收c英镑，现值公式为
```math
PV=\sum_{i=1}^{n}\frac{1}{(1+r)^n}
=\frac{c}{r}
```
>债务的价值向利率的相反方向变化，利率上涨，价值走跌

* 上涨债券
上涨债务价值公式
```math
PV=\frac{c}{r-g}
```
>g必须小于r，否则这个无穷数列就不会收敛

* 年金公式
    在第t个期间内支付c美元
    >房屋抵押就是一种年金
    
```math
PV=c\frac{1-(\frac{1}{1+r})^n}{r}
```
### 预期效用理论

此假说指的是在风险情况下，个人所作出的选择是追求某一数量的期望值的最大化。
效用表示人们对结果的满意度,U(x)。

* 边际效益递减：
    一个人对于财富的占有多多益善，即效用函数一阶导数大于零；随着财富的增加，满足程度的增加速度不断下降，效用函数二阶导数小于零。
* 最大效用原理：
```math
Eu_{x}
```

    在风险和不确定条件下，个人的决策行为准则是为了获得最大期望效用值而非最大期望金额值。

>可以通过系统的风险管理将富豪阶层的资产转移至下层群体
